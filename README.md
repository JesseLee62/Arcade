# Arcade

## Run
Import ArcadeGames.json into database (I use local mongoDB here) 
### Run the following command to install dependencies:
```bash
npm install
```
### To start the application using nodemon or node, run:
```bash
nodemon app.js
```

```bash
node app.js
```
